# 贡献指南

感谢您对ReadWise项目的关注！我们欢迎各种形式的贡献。

## 如何贡献

### 报告Bug
- 使用GitHub Issues提交Bug报告
- 描述Bug的复现步骤
- 提供设备信息和应用版本
- 如果可能，提供截图或录屏

### 提出新功能
- 在GitHub Issues中描述您的想法
- 解释这个功能将如何改善用户体验
- 如果可能，提供UI设计或交互流程

### 提交代码
1. Fork本仓库
2. 创建特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建Pull Request

### 代码规范
- 使用Kotlin进行开发
- 遵循[Kotlin编码规范](https://kotlinlang.org/docs/coding-conventions.html)
- 编写单元测试
- 保持代码注释清晰

## 开发环境
- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Android SDK 34

## 联系我们
如有任何问题，请通过GitHub Issues与我们联系。
